# Linux环境下的CH32V203 makefile最小工程
### 基于 https://github.com/IOsetting/ch32v208-template/ 修改

\
\
\
 
### 开始
克隆仓库并进入最小工程文件夹
```
git clone https://github.com/n0bu9/ch32v20x-makefile-project.git && cd ch32v20x-makefile-project/min_project
```
\
\
### 编译
```
make
```
### 烧录
插上WCH-LINK
```
make flash
```
